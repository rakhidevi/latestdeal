# Design System

**ID:** REQ-UI-001
**Status:** Completed
**Last Updated:** 2026-06-29

## Styling Framework
- **Tailwind CSS (v3+):** Used for rapid, responsive UI development.
- **Compiled Assets:** Tailwind is compiled locally using Vite, and only the minified static CSS is uploaded to production.

## Color Palette
- **Primary:** High-trust Blue (e.g., `#2563EB`)
- **Accent/Sale:** Urgent Red (e.g., `#DC2626`)
- **Background:** Clean Light Gray (`#F3F4F6`) for the feed to make deal cards pop out in pure white (`#FFFFFF`).

## Typography
- **Primary Font:** Inter (Sans-serif) for high readability on mobile devices.
