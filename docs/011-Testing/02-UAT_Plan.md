# User Acceptance Testing (UAT) Plan

**ID:** REQ-TST-002
**Status:** Completed
**Last Updated:** 2026-06-29

## Pre-Launch UAT Checklist
1. **The Shopper Flow:** Can a user visit the site on mobile, filter by 'Electronics', click a deal, and land on Amazon with the Master Affiliate Tag attached?
2. **The Publisher Flow:** Can a test user sign up, add a dummy Telegram Bot token, set a rule for >50% discount, and successfully receive an automated post containing their specific affiliate tag?
3. **The Admin Flow:** Can the admin manually expire a deal and see it instantly removed from the public feed?
