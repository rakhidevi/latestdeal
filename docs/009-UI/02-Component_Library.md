## Reusable Blade Components
To maintain UI consistency and DRY code, the following Blade components (`<x-...>`) will be created:

- `<x-deal-card>`: Displays the product image, title, prices, and CTA.
- `<x-button>`: Primary, secondary, and danger button styles.
- `<x-badge>`: Used for "X% OFF", "Expired", or Category tags.
- `<x-form.input>`: Standardized text inputs with built-in error message display.
- `<x-modal>`: Alpine.js powered modal for confirmations (e.g., "Delete Rule").
